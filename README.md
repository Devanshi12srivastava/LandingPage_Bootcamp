# 🚀 Bootcamp Landing Page

## 📌 Overview

This project is a responsive landing page built using React. It includes a fully functional form integrated with Firebase Firestore to store user data in real-time.



## 🛠️ Tech Stack

* React.js (Vite)
* Tailwind CSS
* Framer Motion
* Firebase (Firestore)

---

## ✨ Features

* Responsive UI (mobile + desktop)
* Smooth animations using Framer Motion
* Form validation (email & phone)
* Duplicate check (email & phone)
* Real-time data storage with Firebase
* Clean and modern design
